Browser History Simulation

A Java-based browser history management system implemented using Stack data structure. It stores visited URLs in Last-In-First-Out (LIFO) order and supports basic browser navigation operations.

Features

* Visit valid URLs
* Store browsing history using Stack
* Remove last two visited pages
* Display current page
* URL format validation

Files

* BrowserHistory.java → Stack operations for browser history
* Main.java → Main class to run the program

Technologies Used

* Java
* Stack
* OOP Concepts

How to Run

1. Compile both Java files
2. Run Main.java
3. Enter URLs and view browser history operations

Output

The program simulates browser history where pages are visited, recent pages are removed, and the current page is displayed.
